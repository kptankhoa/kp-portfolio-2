# kptankhoa.dev

Personal portfolio of Khoa Phan, styled as a retro desktop ("kpOS"): a macOS
Finder-like column browser with a menu bar, window chrome, and a working
terminal. Gruvbox dark palette, Cutive Mono everywhere.

## Stack

Next.js (static export) · React · TypeScript · styled-jsx

## Develop

```bash
npm install
npm run dev      # http://localhost:3000
```

## Build & deploy

```bash
npm run build    # static export to out/
```

Deployed to Cloudflare via Wrangler — see `wrangler.jsonc`.
